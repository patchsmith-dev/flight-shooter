# 星翼突击 Flight Shooter

一个可直接在浏览器运行的飞行射击小游戏。项目使用静态 HTML、CSS、JavaScript 和 Phaser 3，无需构建步骤，适合部署到 GitHub Pages。

## 特色

- 自动射击、键盘移动和指针跟随移动
- 鼠标左键或右键蓄力释放激光炮
- Web Audio 合成背景音乐，以及射击、敌弹、蓄力、爆炸、拾取等音效
- `B` 键或侧栏按钮释放脉冲炸弹，紧急清弹并压制全屏目标
- `Shift` 键或侧栏按钮触发相位冲刺，短暂无敌、加速并清除近身弹体
- 10 个关卡逐步推进，每关 10 波，第 10 波固定 Boss；敌人数量、血量、速度和射击频率逐步提高
- 普通、老兵、噩梦三档难度可切换，难度越高敌人越强，同时得分倍率越高
- 最高分记录、击坠统计、补给进度、连击倍率、本局评级和 Boss 血条
- 连击达到 8、16、28、42 时会触发阶段奖励，补充护盾、炸弹、火力或僚机支援
- 成绩档案会保存最高评级、最高关卡、最高击坠和最高连击
- 非 Boss 小怪 100% 掉落随机道具；每击坠 8 个非 Boss 敌人还会额外触发一次保底补给
- 护盾、火力强化、维修、脉冲清弹、无人僚机和炸弹补给道具；火力与僚机拾取后本局常驻，并可重复拾取升级到上限
- 护盾和维修道具在满值时会扩展护盾容量或生命上限，让防御类补给在后期仍然有价值
- 道具带短标签，靠近战机后会自动吸附，减少错过关键补给的挫败感
- 第 3 波后出现截击机，会横向追踪并发射瞄准弹
- 响应式界面，桌面和移动端都能游玩
- 保留 SVG 素材源文件，运行时使用 Phaser 绘制稳定的 Canvas 纹理

## 本地运行

在项目根目录启动一个静态服务器：

```powershell
python -m http.server 5177
```

然后打开：

```text
http://localhost:5177
```

也可以使用任意静态文件服务器托管本项目。

## GitHub Pages

仓库上传后，可以在 GitHub Pages 中选择 `main` 分支和根目录 `/` 作为发布来源。
